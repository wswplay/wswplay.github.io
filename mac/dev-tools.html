环境变量设置 #

bash
Warning: /opt/homebrew/bin is not in your PATH.
1

默认情况下，并没有把路径进到 bash 中，蛇精病的设计啊。

bash
# 编辑文件
vim .zshrc
# 添加路径
export PATH="/opt/homebrew/bin:$PATH"
# 激活更新文件
source .zshrc
# 搞定
brew -v
# Homebrew 4.1.9

术语解释 #

  • Formulae：软件包。包括了这个软件的依赖、源码位置及编译方法等。
  • Casks：应用包。已经编译好的应用包，如图形界面程序等。
